============
Installation
============

`prnd-notifications`_ 어플리케이션은 Python Package Index ( PyPI )에 등록되어 있지 않습니다. 따라서, git+ssh URI scheme 를 통해서 설치할 수 있습니다.

::

    $ pip install git+ssh://git@github.com/PRNDcompany/prnd-notifications.git


.. _`prnd-notifications`: https://github.com/PRNDcompany/prnd-notifications
